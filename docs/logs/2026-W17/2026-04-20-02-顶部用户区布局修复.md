# Trust Work 平台更新记录

## 版本信息

- 日期：2026-04-20
- 版本说明：右上角用户区布局修复版
- 文档类型：版本更新归档

## 本次更新概览

本次修复聚焦顶部右上角的钱包区与用户区排版。主要解决头像 fallback 文案撑破圆形、用户信息胶囊过高、姓名与角色重复显示导致视觉拥挤，以及右上角三段组件高度不统一的问题。

## 主要更新内容

### 1. 右上角用户区结构收敛

- 保留原有交互职责不变：
  - 头像区域进入个人中心
  - 箭头按钮展开会话菜单
- 用户信息卡由三行压缩为两行：
  - 第一行显示姓名
  - 第二行显示公司，若无公司则回退账号名
- 不再在右上角重复显示角色名称，避免与顶部左上角身份区重复

### 2. 头像 fallback 修复

- 中文姓名 fallback 改为单字展示，避免双字撑坏圆形
- 非中文账号 fallback 保持最多两位大写字符
- 统一头像按钮与 fallback 圆形尺寸，保证圆形视觉稳定

### 3. 顶部对齐与尺寸统一

- 钱包按钮、头像按钮、用户信息卡、箭头按钮统一为更一致的高度体系
- 顶部右侧操作区默认保持单行对齐
- 在移动端仍允许自动换行，避免小屏被挤坏
- 用户信息卡改为更紧凑的宽度与行高，减少顶部压迫感

## 验证结果

- 前端已通过 `npm run build`
- 已确认顶部右侧用户区相关样式已完成重构并成功进入生产构建结果
