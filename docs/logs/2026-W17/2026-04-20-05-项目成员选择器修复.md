# Trust Work 平台更新记录

## 版本信息

- 日期：2026-04-20
- 版本说明：项目管理成员选择器修复版
- 文档类型：版本更新归档

## 本次更新概览

本次修复聚焦管理员“项目管理”页的开发者和客户选择器。主要解决 Windows 原生 `select` 在暗色界面下出现的白底白字、选项可读性差以及展开层与页面视觉风格不统一的问题。

## 主要更新内容

### 1. 项目成员选择器重构

- 将“开发者”和“客户”两个原生 `select` 替换为自定义暗色成员选择器
- 保持原有表单提交流程不变，仍然提交：
  - `developerId`
  - `clientId`
- 选择器收起态直接显示：
  - 姓名
  - 公司或兜底信息

### 2. 下拉菜单视觉修复

- 下拉菜单统一改为暗色浮层
- 选项文本颜色与背景对比恢复正常
- 当前选中项和 hover 项使用高亮状态展示
- 展开层增加最大高度和滚动，避免选项过多时撑坏布局

### 3. 交互补充

- 支持点击空白区域关闭
- 支持按 `Esc` 关闭
- 无可选成员时自动禁用触发按钮

## 影响范围

- 管理员项目管理页的“开发者”选择器
- 管理员项目管理页的“客户”选择器
- 前端新增通用成员选择器组件，便于后续其他成员分配场景复用

## 验证结果

- 前端已通过 `npm run build`
- 已确认项目管理页开发者和客户成员选择器已切换为自定义暗色下拉实现
