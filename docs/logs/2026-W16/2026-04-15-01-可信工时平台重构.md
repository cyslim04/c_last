# Trust Work 平台更新记录

## 版本信息

- 日期：2026-04-15
- 版本说明：角色闭环与平台重构版
- 文档类型：版本更新归档

## 本次更新概览

本次版本围绕“角色闭环演示 + MetaMask 状态完善 + 正常个人中心 + 页面视觉升级”进行了系统性重构。目标是让项目不仅能演示基础流程，还能在答辩或展示场景中清晰体现管理员、开发者、客户三类角色的完整链路。

## 主要更新内容

### 1. 全局界面重构

- 重做整体页面布局，改为侧边导航 + 顶部状态栏结构
- 按角色收敛导航菜单
- 修复原有中文乱码问题
- 整体视觉升级为更强展示感的答辩风格
- 增加玻璃面板、渐变背景、悬浮动效与页面进入动画

### 2. MetaMask 交互升级

- 连接成功后，按钮显示为账户缩写和当前网络
- 支持监听账户切换和网络切换
- 支持在 `Sepolia` 与本地 `Hardhat` 之间快速切换
- 支持切换账户、切换网络、断开连接
- 钱包地址变更后，自动同步到当前登录用户资料

### 3. 个人中心与认证完善

- 新增独立个人中心页面
- 支持修改姓名、邮箱、手机号、公司/团队、个人简介
- 支持头像上传
- 支持修改密码
- 支持退出登录
- 后端新增 `me/profile/password/avatar/logout` 相关接口

### 4. 后端逻辑与权限完善

- 新增角色权限控制
- 管理员可创建项目和查看审计
- 开发者只可处理自己参与项目的工时和交付物
- 客户只可查看并确认自己项目下的交付物
- 列表接口补充了项目名、成员名等展示字段
- 项目创建由输入 ID 改为用户下拉选择

### 5. 演示数据重建

- 引入 `app_meta.seed_version` 机制
- 数据库为空或版本不匹配时自动重建完整演示数据
- 默认重建以下内容：
  - 3 个基础账号
  - 2 个演示项目
  - 多条工时记录
  - 多条交付物记录
  - 钱包绑定
  - 审批记录
  - 审计日志
  - 存证回写记录

### 6. 审计与流程闭环

- 控制台支持按当前角色查看核心指标与最近记录
- 审计页可查看：
  - 最近交易回写
  - 操作日志
  - 项目闭环统计
- 客户确认页支持填写审批意见并通过/驳回

## 新增或调整的关键接口

- `POST /api/auth/login`
- `GET /api/auth/me`
- `PUT /api/auth/profile`
- `PUT /api/auth/password`
- `POST /api/auth/avatar`
- `POST /api/auth/logout`
- `POST /api/auth/wallet`
- `GET /api/users?role=...`
- `GET /api/dashboard`
- `GET /api/audit`
