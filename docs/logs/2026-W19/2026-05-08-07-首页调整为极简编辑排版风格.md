# 首页调整为极简编辑排版风格

## 变更概述
- 按照最新首页参考效果，将首页从“多卡片系统展示”进一步收敛为“极简编辑排版”结构。
- 保留可信工时、交付物确认、客户确认、阶段审计与最终审计的系统叙事，但改用更强的版式节奏和更少的装饰来承载内容。
- 首页信息仍然聚焦系统信息、当前功能与平台资料，不回退到旧控制台式布局。

## 具体调整
- 重写 [frontend/src/views/HomeView.vue](/C:/Users/yce/Desktop/c_last/frontend/src/views/HomeView.vue)，将首页改为五段式主板面结构：
  - 黑色首屏承载系统主张与可信能力标签
  - 平台角色区按管理员、开发者、客户三栏展开
  - 协作流程区用四段分栏说明闭环
  - 技术档案区以系统资料为主体，并在侧栏压缩展示可信能力与状态摘要
  - 底部深色行动区承接当前登录身份与主操作入口
- 更新 [frontend/src/styles/main.css](/C:/Users/yce/Desktop/c_last/frontend/src/styles/main.css)，新增首页专用视觉变量与排版样式，整体收紧为细线分栏、大留白、低饱和背景和小面积强调色。
- 保留 `IntersectionObserver` 驱动的章节 reveal，并在 `prefers-reduced-motion` 下继续抑制动画。
- 保留 `aria-live="polite"` 错误反馈与主要交互的 `focus-visible`，继续满足 `web-design-guidelines` 的可访问性要求。
