# 首页改为可信工时与交付物系统展示页

## 变更概述
- 将首页从原来的功能拼盘式布局改为系统能力展示页。
- 首页主角改为“区块链下工时与交付物的可靠性”，弱化实时业务数据的主舞台地位。
- 维持现有简洁、收敛、企业感的视觉基调，同时增强排版层级、章节节奏和首页记忆点。

## 具体调整
- 重写 `frontend/src/views/HomeView.vue`，将内容重组为系统主张、核心能力、技术档案、角色功能和轻量状态摘要五个部分。
- 保留 `GET /api/dashboard` 与 `GET /api/process-flow` 作为首页唯一数据来源，不新增后端接口。
- 新增技术档案区，用平台级资料替代个人资料展示，强调区块链作用、存证对象、角色结构与可追踪链路。
- 将功能展示方式改为按管理员、开发者、客户三组解释职责与入口，不再按菜单平铺。
- 在 `frontend/src/styles/main.css` 中增加新版首页的布局、卡片、概念流程、响应式和焦点态样式，并同步清理旧首页残留规则。
- 结合 `web-design-guidelines` 做了首页级可访问性收口，补充错误信息 `aria-live` 与主要入口的 `focus-visible` 反馈。

## 验证
- 已在 `frontend/` 执行 `npm run build`，构建通过。
