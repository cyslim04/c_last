# 顶部导航与首页重构

## 变更概述
- 将登录后默认入口改为 `/home`，`/` 与旧 `/dashboard` 统一兼容跳转到首页。
- 将全局壳层从左侧侧栏改为顶部统一导航，桌面端与移动端共享同一套导航信息架构。
- 新增登录后首页，整合原总览页数据、流程高层结构说明、当前状态摘要与按角色导流入口。

## 具体调整
- 重写 `frontend/src/components/AppShell.vue`，保留钱包连接、会话菜单和主题切换，移除旧侧栏布局逻辑。
- 新增 `frontend/src/views/HomeView.vue`，复用 `/api/dashboard` 和 `/api/process-flow` 的现有数据，不新增后端接口。
- 更新 `frontend/src/router/index.js` 和 `frontend/src/views/LoginView.vue`，确保登录后和兼容路径都进入首页。
- 将 `frontend/src/views/DashboardView.vue` 改为纯前端兼容跳转页，避免旧“控制台”入口继续承载真实 UI。
- 扩展 `frontend/src/styles/main.css`，新增顶部导航、首页章节、滚动 reveal 和响应式规则，同时清理旧侧栏交互残留。

## 验证
- 已在 `frontend/` 执行 `npm run build`，构建通过。
