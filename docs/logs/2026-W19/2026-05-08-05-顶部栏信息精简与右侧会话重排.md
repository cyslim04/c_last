# 顶部栏信息精简与右侧会话重排

## 变更概述
- 删除顶部栏中部的角色与平台说明块，避免头部信息堆叠。
- 将顶部栏重新整理为“左侧品牌 + 中部导航 + 右侧钱包与个人操作”的结构。
- 保持移动端顶部展开导航逻辑不变，同时让个人操作继续稳定落在最右侧。

## 具体调整
- 更新 `frontend/src/components/AppShell.vue`，移除中部身份说明区域，仅保留品牌、导航、钱包和会话入口。
- 更新 `frontend/src/styles/main.css`，重新分配顶部栏主区与操作区宽度，让钱包区贴近右侧、个人操作位于最右端。
- 保留原有钱包浮层、用户菜单、主题切换和退出登录能力，不改业务行为。

## 验证
- 已在 `frontend/` 执行 `npm run build`，构建通过。
