# 全局配色统一并移除主题切换

## 变更概述
- 将整套前端界面的全局配色统一到首页当前使用的黑 / 米白 / 砖红体系。
- 删除亮暗色双主题切换能力，避免首页与其他页面之间视觉语言不一致。
- 同时收掉按角色切换的蓝色、绿色、金色强调色，统一使用同一套砖红强调色。
